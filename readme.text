# 💬 Real-Time Chat App

A real-time web chat application built with **Node.js**, **Express**, **React**, and **WebSockets (Socket.io)**. It supports multi-user messaging with live updates and a responsive interface.

---

## ⚙️ Tech Stack

- **Frontend**: React
- **Backend**: Node.js + Express
- **WebSockets**: Socket.io
- **Styling**: CSS / Tailwind / Bootstrap (optional)
